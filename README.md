# ReactOS-Google-Balls-Edition
ReactOS Google Balls Edition: Small. Fast. Elegant.

Why bother with clunky and outdated operating systems for your computing like Linux® or Windows® when ReactOS: Google Balls Edition covers all of your computing needs perfectly!

No updates, no internet, EXTREMELY LIGHTWEIGHT AND FAST, PRODUCTION READY, NO SOCIAL MEDIA. Perfect for anyone including terminally online people and brainrotted indiviuals!

System Requirements

- A old 32-bit x86 processor (You probably have one) which usually comes on hardware made before 2005. No x64 here due to NTOSKRNL.EXE BSoD.
- 24 bit color depth capable video card (Comes on most Windows® XP era computers and later)
- PS/2 or USB keyboard and mouse
- 128 megabytes of memory

How to use

- Turn on computer
- Type "launch" into the command line
- Use the mouse to move the cursor and watch the Google Balls react for a few minutes or hours
- Turn off computer
- Go outside
